NetFPGA-Related Paper-Overview 2013-2017

2017.04.29 Wasdns  热度 ℃

1.Implementation of TCP large receive offload on open hardware platform

propose: open hardware platform of a stateless Large Receive Offload method (LRO)

target: receiver processing power

接收方处理数据的能力

2.From 1G to 10G: code reuse in action

propose: new, flexible, legacy support mechanics for designs built using System on a Chip (SoC) and System on FPGA (SoFPGA).

target: to make existing, systems reusable in this new, higher-speed scenario

代码兼容性

3.PiBuffer: 面向数据中心的OpenFlow流缓存管理模型

propose: 支持流内报文保序的OpenFlow交换机流缓存管理模型。

target: 缓解集中的控制平面，减少冗余packet-in次数，避免流内报文乱序。

减轻CP处理压力

4.A Software Defined Sketch System for Traffic Monitoring

propose: a scheme for the sketch-based monitoring applications

target: the flexibility of adopting the same data structure for various
traffic measurement application

流量监控

5.Superspreader Detection System on NetFPGA Platform

propose: a system to detect superspreaders

target: estimate their fan-outs accurately using small amount of memory

安全 攻击源发现

6.Blueswitch: Enabling Provably Consistent Configuration of Network Switches

propose: a hardware design that supports a transactional configuration mechanism, and provides packet-consistent configuration.

target: consistent configuration updates => even simple rule updates result in inconsistent packet switching in multi-table datapaths.

流表更新的兼容性

7.Enabling Performance Evaluation Beyond 10 Gbps

propose: OSNT, a community-driven, high-performance, open-source traffic generator and capture system

target: network monitoring and testing => cheap and flexiable

收发包系统，网络监测、测试

8.Flexible, extensible, open-source and affordable FPGA-based traffic generator

propose: a traffic generator capable of filling a 10 Gb/s Ethernet link, with traffic features specified in software

target: suitable for high-speed links; generate traffic with some specified features; cheap

流量生成器，网络测试

9.Architecture for an open source network tester

key words: monitoring, netfpga, open source, packet-sniffing, programmable hardware, traffic-generation, packet-capture, high speed

流量生成器，抓包工具

10.Millions of little minions: using packets for low latency network programming and visibility

propose: a practical approach to rapidly introducing new dataplane functionality into networks; “tiny packet program” (TPP)

target: visibility into network behavior

网络行为的可视化能力
